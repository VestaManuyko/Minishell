______Errors______

In case of system error:
perror("Error\n[Function name]");
In function error:
write(2, "Error\n[Function name] and error_message", length);

____File_names____

Names of the functions includes to minishell.h to be named after what they handle.
Example:
-arr_deepcpy.c
-arr_free.c
-arr_printf.c
